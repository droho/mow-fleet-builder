# Man O' War Fleet Builder

Fleet builder for the Man O’ War tabletop game based on the Community Rulebook.

## Features
- All official fleets (including Undead)
- Real-time validation (live checks)
- Fleet analysis tools
- Unit preview
- Scenario generator
- EN / PL language support
- Mobile-friendly UI
